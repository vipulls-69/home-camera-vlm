## Reporting a Vulnerability
Please do not report security vulnerabilities through public GitHub issues, discussions, or pull requests. If you discover a security vulnerability within this project, please report it privately to ensure it can be patched before becoming public knowledge.

How to Report
Email: Send an email to sharmavipul01002@gmail.com with the subject line [SECURITY] Vulnerability Report: <Brief Description>.
